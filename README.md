## 💻 Sobre o projeto
  Exercícios de Objetos e Funções do Javascript, feitos pela <a href="https://www.cubos.academy/" target="_blank">Cubos Academy</a>, com a minha resolução:

1. Sistema de correção de provas
2. Ligar, desligar, acelerar e freiar um Carro

## 🚀 Como executar o projeto

```bash
# Clone este repositório
$ git clone https://github.com/MayllaRabay/objetos-funcoes.git

# Acesse a pasta do projeto no terminal/cmd
$ cd objetos-funcoes

# Acesse cada exercício individualmente para solucioná-lo (leia antes seu respectivo arquivo README)
$ cd exercicio-xx

# Assim que você terminar sua resolução na pasta index.js, abra o terminal e execute o comando:
$ node index.js
```
  Lembre-se de usar o **console.log()** para a sua resposta aparecer no terminal.

  Teste sua resposta com diferentes inputs e bons exercícios!
